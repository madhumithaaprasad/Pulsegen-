# Output Reports

This folder contains sample output reports generated by the system after processing customer feedback data.

## trend_report.csv
- Rows represent normalized, business-ready topics
- Columns represent dates from T-30 to T
- Each cell contains the frequency of the topic on that date
- This report enables trend analysis of customer issues, requests, and feedback

Note: These files are sample outputs generated using test data to demonstrate the expected structure and format.
